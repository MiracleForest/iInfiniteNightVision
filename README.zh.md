# iInfiniteNightVision

[English](README.md)

这是一个用于 LeviLamina 模组加载器的 Minecraft 无限夜视插件。

## 说明

这个插件会让客户端画面持续保持较高亮度，模拟类似夜视的视觉效果，从而让黑暗区域更容易看清。

现在还支持水下夜视，进入水下后视野也不会再受限。

这是一个仅支持客户端的插件，不支持服务端安装或使用。

## 使用方法

安装并由 LeviLamina 加载后，亮度效果会自动生效，包括水下场景。

正常使用时不需要额外指令，也不需要额外配置。

## 安装

> [!CAUTION]
> 本插件仅支持客户端安装。

### 客户端

#### 使用 Lip 安装（推荐）

1. 进入你的 Minecraft 版本目录，例如 `versions/<你的版本>/`
2. 执行以下命令：
   ```bash
   lip install github.com/MiracleForest/iInfiniteNightVision#client
   ```

#### 手动安装

1. 从 [Releases 页面](https://github.com/MiracleForest/iInfiniteNightVision/releases) 下载 `iInfiniteNightVision-windows-x64.zip`
2. 打开 LeviLauncher，进入模组管理页面
3. 将下载好的 ZIP 文件拖入模组页面完成安装

**注意：** 使用本插件前，请确保客户端已经安装 LeviLamina。
